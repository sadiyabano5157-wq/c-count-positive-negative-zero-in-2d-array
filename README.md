# c-count-positive-negative-zero-in-2d-array
A C program to count positive, negative, and zero elements in a 3×3 two-dimensional array using nested loops and conditional statements.

# Count Positive, Negative and Zero Elements in 2D Array (C)

## Description
This project counts the number of positive, negative, and zero elements present in a 3×3 two-dimensional array.

## Concepts Used
- C Programming
- 2D Arrays
- Nested Loops
- if-else Statements
- Counters
- Conditional Logic

## Array
0   -20   30
-40  50  -60
70  -80   90

## Output
Positive number count = 4
Negative number count = 4
Zero number count = 1
